Para realizar a consulta de Clientes / Pedidos, selecione no menu Consulta a opção "Clientes". A tela de consulta de Clientes será aberta.

Deve-se escolher os parâmetros necessários para as consultas.
Pode-se escolher apenas ou parâmetro ou um conjunto de parâmetros, após selecionar os filtros para a consulta clicar no botão Pesquisar, o resultado da consulta será exibido na tela.

Se necessário gerar um relatório ou imprimir o resultado da consulta deve-se clicar no botão Relatório para ser gerado um documento que poderá ser impresso.

Barra de Botões da Consulta:

Botão Pesquisar, após escolher os parâmetros das consultas clicar no botão Pesquisar para visualizar os resultados.

  Botão Relatório, após realizar a pesquisa e visualizar os resultados na tela, clicar no botão Relatório para gerar um Relatório em uma folha que pode ser impressa.

Botão Voltar para tela Inicial.
